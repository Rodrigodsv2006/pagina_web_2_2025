Fundamentos de Ingeniería Informática (FII)

Descripción general

Esta materia es la piedra angular del grado. Cubre desde el nivel más bajo (puertas lógicas) hasta la interacción del sistema operativo.

Contenidos:

      PARTE I: FUNDAMENTOS TEÓRICOS

    • BLOQUE 1. La ingeniería Informática en la Sociedad
    • BLOQUE 2. Paradigmas actuales y tendencias de los sistemas y las aplicaciones
    • BLOQUE 3. La información y los datos

      PARTE II: HABILIDADES PRÁCTICAS: INTRODUCCIÓN AL DESARROLLO WEB

    • BLOQUE 1. Introducción al desarrollo de Software
    • BLOQUE 2. Creación de páginas Web HTML5
    • BLOQUE 3. Diseño y maquetación con CSS3

Objetivos:

  • Aprender a realizar páginas web y dominar HTML y CSS
  • Aprender representación de datos (Binario, Hexadecimal).
  • Dominar los conceptos básicos de redes y protocolos.
